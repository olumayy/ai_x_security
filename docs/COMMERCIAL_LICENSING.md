### Can I use this for a free community workshop?

**Yes, with conditions**:
- If you're an individual or non-profit, you can run free workshops with proper attribution
- If you're a for-profit company hosting free workshops (marketing/lead generation), you need a corporate license

### What if I'm already using it without a license?

**No problem!** We understand licensing can be confusing. Simply:
1. [Contact me](https://www.linkedin.com/in/raymond-depalma/) to discuss retroactive licensing
2. Most cases can be resolved with a discounted rate for the first year

### Do I need a license if I just reference your work in my blog?

**No.** Blog posts, tutorials, and personal content that references or builds upon this work are permitted under CC BY-NC-SA 4.0 as long as:
- You provide attribution
- You're not charging for access to that content
- You link to the original repository

### Can I fork the repository for my company's internal use?

**Forking is allowed**, but:
- If using commercially (corporate training), you need a license
- Must maintain attribution and license notices
- Must comply with CC BY-NC-SA 4.0 ShareAlike terms for any modifications

### What's included in "quarterly updates"?

- New threat actor TTPs (APT groups, ransomware families)
- Updated AI model references (Claude, GPT, Gemini versions)
- New detection techniques and tools
- Bug fixes and improvements
- Additional labs (as we create them)

### Can I white-label this for my brand?

**No.** Attribution to the original work is required. However:
- Training Provider licenses include co-branding options
- You can add your branding alongside attribution
- Custom enterprise licenses may have different terms

### What if I want to translate this into another language?

**Great!** Translation falls under derivative works (CC BY-NC-SA 4.0):
- Personal/academic use: Free with attribution
- Commercial use: Requires commercial license
- Reach out if you're interested - translations help the community!

### Do you offer multi-year discounts?

**Yes!**
- 2-year commitment: 10% discount

- Available for Corporate and Training Provider licenses
